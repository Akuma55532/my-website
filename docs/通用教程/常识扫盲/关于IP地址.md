# 关于IP地址

IP地址可以理解为设备在网络中的“门牌号”。电脑、路由器、树莓派、无人机等设备，只有知道彼此的IP地址，才能在同一个网络中互相通信。

## IP地址

常见的IP地址一般写成 `192.168.1.10` 这样的形式。

- 每一台联网设备都需要一个IP地址
- 在同一个局域网内，IP地址通常不能重复
- 如果地址重复，设备之间就可能无法正常通信

## 子网掩码

子网掩码用于判断哪些设备和自己属于同一个局域网，常见写法是 `255.255.255.0`。

例如：

- 设备A：`192.168.1.10`
- 设备B：`192.168.1.20`
- 子网掩码：`255.255.255.0`

这种情况下，它们通常属于同一个局域网，可以直接互相访问。

## 网关

网关可以理解为设备访问外部网络时要经过的“出口”，通常就是路由器的地址。

例如：

- 设备IP：`192.168.1.10`
- 网关：`192.168.1.1`

当设备需要访问其他网段或互联网时，一般会先把数据发给网关。

## 静态IP

静态IP就是手动设置的固定IP地址。

- 重启后地址通常不会变化
- 适合需要长期固定连接的设备

例如把树莓派设置为 `192.168.1.50`，以后就可以一直通过这个地址访问它。

## DHCP服务

DHCP是一种自动分配IP地址的服务，家用路由器通常都带有这个功能。

它会自动给设备分配：

- IP地址
- 子网掩码
- 网关

这样就不需要每台设备都手动设置，使用起来更方便。

## 简单总结

- `IP地址`：设备的门牌号
- `子网掩码`：判断是否在同一局域网
- `网关`：访问外部网络的出口
- `静态IP`：手动设置的固定地址
- `DHCP`：自动分配网络参数的服务
